# Hello, World

If you're reading this, the site is live. Welcome.

I've been meaning to start writing for a while now. Not the "I should really blog more" kind of meaning — the kind where you keep a running list of things you wish someone had written about when you were trying to figure them out at 2 AM.

## What to Expect

This blog will mostly cover things I'm working on or thinking about:

- **Distributed systems** — the patterns, the gotchas, and the "why did nobody warn me about this" moments
- **ML in production** — not the Jupyter notebook kind, the "this needs to run at scale and not lose money" kind
- **Building in public** — lessons from working on [GlowLocal](https://glowlocal.co) and other side projects

> **Note:** If you're here from my LinkedIn or GitHub — thanks for checking this out. I'll try to keep things practical and skip the fluff.

## The Stack

This site is built with [Astro](https://astro.build), styled with Tailwind CSS v4, and deployed on Vercel. Blog posts are written in MDX, which means I can drop in interactive components when it makes sense.

The source is intentionally simple. No CMS, no database, no over-engineering. Just markdown files in a repo — commit and deploy.
